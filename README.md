# Personal Homepage

这是一个适合部署到 GitHub Pages 的静态个人主页。

## 文件说明

- `index.html`: 页面内容结构
- `styles.css`: 页面样式
- `script.js`: 轻量滚动动效

## 你接下来要改的地方

1. 把 `index.html` 里的名字、简介、项目、邮箱、GitHub 链接改成你自己的。
2. 如果你想换颜色或风格，优先改 `styles.css` 里的 `:root` 变量。
3. 如果你有头像，可以把 `.portrait` 换成真实图片区域。

## GitHub Pages 发布方式

1. 在 GitHub 创建一个仓库。
2. 如果这是你的 GitHub 个人主页，仓库名建议使用 `你的用户名.github.io`。
3. 把这个文件夹内容上传到仓库根目录。
4. 在仓库的 `Settings -> Pages` 中开启 GitHub Pages。
5. Source 选择对应分支（通常是 `main`）和根目录 `/root`。

发布后，页面通常会出现在：

`https://你的用户名.github.io/`
